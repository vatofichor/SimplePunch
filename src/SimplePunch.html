SimplePunch

Zero-dependency browser punch clock

Time: 0.00 hrs
# Type Timestamp
Select a folder and file to begin.
  1. 1 Load Folder — Click 📁 Load Folder and pick the directory on your machine where punch files are stored. This grants the app read/write access for that session.
  2. 2 Select a file — The dropdown populates with all .txt files in that folder. Pick the one you want to work with. The punch log will load immediately.
  3. 3 Create a new file — Click + New File to create a fresh log named with today's date (e.g. 2026-06-19.txt). After creating, click 📁 Load Folder again so the new file appears in the dropdown.
  4. 4 Clock In / Clock Out — Use the ⬇ Clock In and ⬆ Clock Out buttons to record punches. Each punch is written directly to the file. Total hours update live.
  5. 5 Edit a timestamp — Click the ✏ pencil icon on any row to correct a timestamp inline. Press Enter or ✓ to save, Escape or ✕ to cancel. Format: YYYY-MM-DD HH:MM:SS.
  6. 6 Delete an entry — Click the 🗑 trash icon to remove a row. The file is rewritten immediately. Use this to correct mistakes — order matters for accurate totals.
⚠ Reload after file changes. This app is serverless — the folder dropdown is a one-time snapshot. After creating a new file or switching to a different folder, click 📁 Load Folder again to refresh the file list.
